# Reservation System UI

A modern React-based user interface for a universal reservation and scheduling management system.

---

## Tech Stack

- **React 19** - Modern JavaScript library for building user interfaces.
- **Vite 7** - Next-generation frontend build tool for fast development.
- **JavaScript (ES2023+)** - Core programming language (Node.js 23.8).
- **Custom CSS** - Hand-crafted styles without external frameworks.
